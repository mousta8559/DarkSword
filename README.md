## 🧭 What DarkSword Is

DarkSword is a Windows app for working with exploit chain dumps in a simple way. It helps you open, inspect, and review dump files without needing to use command-line tools.

Use it if you want a local tool to look through chain data on your PC. It runs on Windows and keeps the setup process simple.

## 🛠️ Troubleshooting

### DarkSword does not open

- Check that the download finished
- Make sure you extracted the ZIP file first
- Try running the app from the extracted folder
- Right-click the file and choose Run as administrator

### Windows blocks the app

- Open the release page again
- Download the file from the latest release
- Confirm that you picked the Windows asset
- Try the file again after extraction

### My file will not load

- Check that the file is a supported dump file
- Try another export from the same source
- Make sure the file is not empty
- Move the file to a local folder on your PC

### The app feels slow

- Close other apps that use a lot of memory
- Use smaller files first
- Avoid opening several large dumps at the same time
- Restart DarkSword and try again
